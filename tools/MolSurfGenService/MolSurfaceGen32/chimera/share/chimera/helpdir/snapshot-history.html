UCSF Chimera Snapshot Remarks

Not all improvements are listed.

Release 1.1864 - 10/6/2003

First snapshot release. Contains many bug fixes and functionality enhancements.

  • Multiscale extension added. Displays large complexes like viruses and ribosomes.
  • UnitCell extension added. Loads copies of molecules to fill the crystallographic unit cell.
  • ConstrainedMove extension added. Provides mouse mode for rotating about a specified axis.
  • UndoMove extension added. Keeps a history of model positions. Useful for correcting mistakes when hand aligning models.
  • "Save image" does supersampling. This smooths jagged edges especially noticable on white backgrounds.
  • Per model clip planes. Each model can have its own clip plane with any orientation, adjustable with mouse. Use the Model Panel / Clipping button.
  • Faster isosurface calculation. About 10 times faster for 128^3 volumes on Windows, and 3 times faster on other platforms.
  • Faster volume surface/mesh rendering on NVidia GeForce 4 Ti graphics. About 3 times faster.
  • SPIDER volume file reader added. Electron microscope density map format.
  • BRIX volume file reader added. Density map format crystallography and electron microscopy.
  • DOCK 4 scoring grids volume file reader added.
  • Accelerators added to Accelerators extension. More menu entries now have 2 letter accelerators.
  • Line thickness control for volume meshes added. Good for high resolution images.
  • Non-saturating transparency mode for volumes added. Permits transparency with dim specular highlights.
  • Fixed Phantom force feedback extension. It was broken in the 1.1700 production release.
