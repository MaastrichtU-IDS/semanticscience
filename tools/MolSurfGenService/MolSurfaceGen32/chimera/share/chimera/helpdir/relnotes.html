Chimera Release Notes

All releases with the same version number have the same feature set across all supported platforms. There will be some visual differences due the the different windowing systems used. Please let us know if you notice a behavioral difference.

You can see which version of Chimera you are running by choosing About UCSF Chimera from the Help menu.

Release notes are added as new versions are released. The current release history and release notes can be viewed at http://www.cgl.ucsf.edu/chimera/docs/relnotes.html.
