Release History

UCSF Chimera Version 1.0 Build 2184 Release Notes (31 October 2005)

Changes since the previous production release (1.0 build 2180/2181):

Major Bugs Fixed:

  • On UNIX, using help no longer hangs chimera when there was no web browser already running
  • On Windows, saving images in formats other than PNG works again (PNG is still the best format for most uses)
  • Creating large JPEG images now works
  • Surface Caps no longer confuse Close Session

General:

  • new input format, BILD: simple descriptions of lines, polygons, and basic geometric forms are used to construct a VRML model

Known Bugs:

  • BILD format documentation is missing
  • On Mac OS X 10.2, 3d texture volumes don't work

For Developers:

  • inplace operators (+=, etc.) work again for Vector, et. al.
