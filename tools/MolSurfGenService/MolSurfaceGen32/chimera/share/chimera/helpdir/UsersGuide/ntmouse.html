Windows - Middle Mouse Button Workaround

The default Chimera middle mouse button assignment of XY-translation may not work on Windows systems using Logitech MouseWare. The middle mouse button is set to perform autoscrolling and does not get reassigned automatically to XY-translation within Chimera. To solve this, change the Logitech setting for the middle mouse button from Universal Scroll to Generic Button.
