Model Attributes: VRMLModel and Volume_Model

Once a model has been chosen in the left side of the Model Panel, clicking the attributes... button in the right side of the panel brings up a window that shows and allows changes in model attribute values.

Models of class VRMLModel:

  • MS/DMS surfaces
  • PipesAndPlanks representations
  • objects read from BILD files
  • objects read from VRML files
Models of class Volume_Model:
  • Volume Viewer transparent solid displays
The attributes of models in these classes are:

  • active - whether the model is activated for motion
    • false
    • true
  • displayed - whether display is enabled at the model level (see display hierarchy)
    • false
    • true
