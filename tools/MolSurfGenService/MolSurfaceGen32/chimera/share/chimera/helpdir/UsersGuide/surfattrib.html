Model Attributes: SurfaceModel and Volume

Once a model has been chosen in the left side of the Model Panel, clicking the attributes... button in the right side of the panel brings up a window that shows and allows changes in model attribute values.

Models of class SurfaceModel are created by:

  • Icosahedron Surface
  • Intersurf, command intersurf
  • Multiscale Models
  • Surfnet
  • Thermal Ellipsoids, command aniso
  • Volume Tracer
  • other commands: shape, topography, hkcage, measure inertia, etc.
  • reading in certain files: IMOD surfaces, GRASP surfaces from DelPhi or GRASP
In addition, specialized subtypes of SurfaceModel are:
  • class Volume – Volume Viewer contour surfaces (surface or mesh)
  • class MSMSModel – molecular surfaces
The attributes of models in these classes are:

  • active - whether the model is activated for motion
    • false
    • true
  • displayed - whether display is enabled at the model level (see display hierarchy)
    • false
    • true
  • one transparent layer - whether to show only the topmost layer when the surface is transparent
    • false
    • true
  • pieces - number of surface pieces (independently selectable subparts) in the surface model. A surface model may consist of one or more surface pieces, and a single surface piece may consist of one or more blobs (disconnected parts). Surface piece attributes are shown in the Selection Inspector.
  • selectable with mouse - whether the surface piece(s) comprising the model can be picked with the mouse
    • false
    • true
  • triangles - number of triangles in the surface model
