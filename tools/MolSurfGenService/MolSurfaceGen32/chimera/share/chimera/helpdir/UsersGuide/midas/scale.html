Chimera Commands Index

Usage:
scale factor [ frames ]

Usage:
~scale

Scale changes the size of the displayed view by the specified scaling factor. The scaling factor must be positive; a scaling factor less than one will decrease the size of the displayed view. Note that this command does not modify the model coordinates.

Scale changes the size of the view by factor for the specified number of image update frames (default 1). ~Scale will halt an ongoing scaling process. Commands continue to be processed while the requested motion is in progress. To pause command processing until the motion is finished, use the wait command.
