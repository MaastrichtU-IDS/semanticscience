Instances of Tools

Instances of certain tools and data tables are available from the Tools menu. A previously started tool or previously generated data table may be listed in the bottom section of the Tools menu:

Tools... Instance_name
where Instance_name includes the name of the tool or table and sometimes the name of the data that it shows or manipulates. There can be multiple instances of the same tool with the same data or with different data. A submenu for each instance contains the options:
  • Raise - show the interface
  • Hide - hide the interface without exiting from the instance
  • Quit - exit from this instance
