Chimera Commands Index

Usage:
namesel [ name ]

Namesel names and saves the current selection. This can also be done with Select... Name Selection in the menu. The selection contents are remembered, not the process by which the selection was generated (i.e., even if all oxygens in the current structures are selected, the named selection will not include oxygens in different structures opened later).

The namesel command without any arguments lists the names of all saved selections in the status line and Reply Log.

See also: alias, select
