Chimera Commands Index

Usage:
clip plane distance [ frames ]

Usage:
~clip plane

Clip moves a global clipping plane a specified distance from its current position. The plane is specified as hither (front) or yon (back). The distance is a value in the current display units, usually Å; a positive number moves the plane towards the user, while a negative number moves it away from the user.

Clip moves the clipping plane in the specified manner for the specified number of image update frames (default 1). ~Clip halts an ongoing clip plane motion. Commands continue to be processed while the requested motion is in progress. To pause command processing until the motion is finished, use the wait command.
