Chimera Built-in Colors

Chimera includes many named colors:

  • all 60 [top 20]
  • 109 element colors
There is some overlap; for example, the names white and H refer to the same color. The element symbols can be used just like any other color names.

Additional colors can be created and named with the command colordef, or created interactively and saved to a palette with the Color Editor.

Color names are used in coloring commands (color, modelcolor, rangecolor, etc.) and can be entered directly into the Color Editor.

In built-in color names, gray and grey can be used interchangeably.

Element Colors

These are the same as the Jmol element colors. Mousing over an entry shows the corresponding Tk color code.
