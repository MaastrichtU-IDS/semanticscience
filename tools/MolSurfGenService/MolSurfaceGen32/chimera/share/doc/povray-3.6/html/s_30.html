1.5.1 POV Files

POV-Ray for Unix works with two types of plain text files. The first is the standard POV-Ray scene description file. Although you may give files of this type any legitimate file name, it is easiest if you give them the .pov extension. In this Help file, scene description files are referred to as POV files.

The second type, the initialization file, is new to POV-Ray 3. Initialization files normally have .ini extensions and are referred to in this help file as INI files.
