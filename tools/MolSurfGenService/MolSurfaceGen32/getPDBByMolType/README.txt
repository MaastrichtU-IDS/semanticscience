
To get all RNA pdb codes run the following:
php getPdbByMolType.php -protein N -dna N -rna Y -hybrid N



To get all Protein pdb codes run:
php getPdbByMolType.php -protein Y -dna N -rna N -hybrid N


To get all Protein and DNA pdb codes run:
php getPdbByMolType.php -protein Y -dna Y -rna N -hybrid N
